# althire

### Prerequisites

- bun.js
- Docker Compose

### Get Started

- Install dependencies 
```bash
bun install
```
- Run the development server
```bash
bun dev
```
- Seeding locations
```bash
wget https://download.geonames.org/export/dump/IN.zip
wget https://download.geonames.org/export/dump/US.zip
unzip IN.zip
unzip US.zip
```
### Extras

- Upgrade Next.js
```bash
bun install next@latest react@latest react-dom@latest
```
